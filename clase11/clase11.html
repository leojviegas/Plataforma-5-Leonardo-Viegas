METODOS

sirven para hacer cosas con los arrays

METODO: push

sirve para agregar un elemento al final de array, y te devuleve el lenght actualizado del array

comidas.push("ravioles")

METODO: pop

sirve para quitar un elemento al final de array y lo va a retornar

comidas.pop()

METODO: unshift

añadir elementos al array, pero al principio

METODO: shift

quitar elementos al array, pero al principio

METODO: indexOf

busca si hay un elemento (que vos el establezcas) en el array. Si lo encuentra, te devuleve la posicion del mismo. Si no lo encuentra, te devuelve -1

METODO: slice

sirve para crear una copia de nuestro arreglo original y este no se verá afectado

si NO le ponés parametros, te hace una copia exacta. si le ponés 2 parámetros( ej: 0, 3) te va a hacer una copia solamente con la franja de elementos que le seleccionaste, excluyendo el último

METODO: splice

sirve para SACAR,AGREGAR O REEMPLAZAR elementos a un array

let ejemplo.splice(param1, param2, param3 opcional)

  1. parametro 1: la posición a la que querés apuntar
  2. parametro 2: la CANTIDAD de elementos hacia la deracha que querés sacar, inclusive el de la posicion en la que estás
  3. parametro 3 en adelantes -opcionales-: si querés agregar un elemento en la posicion en la que estás apuntando con parámetro 1, podés agregarlo acá

let arrayHola = ["hola", "hello", "konichiwa", "oi", "ciao"]

let arrayHola.splice(1, 2)

lo de arriba va a sacar "hello", y "konichiwa", y devolver eso. ahora arrayHola contiene ["hola", "oi", "ciao"]

let arrayHola.splice(1, 0, "Salut")

lo de arriba va a agregar "Salut" en la posicion 1 (donde ahora está "oi", sin removerlo), y lo demás se mueve a la derecha

METODO: join

sirve para transformar ARRAY en un string

METODO: split

al revés que join, sirve para transformar un string en un ARRAY

IMPORTANTE PARA EL TRADUCTOR DE ROSARIGASINOOOOO

como parámetro, se pone LO QUE QUERÉS QUE IDENTIFIQUE COMO SEPARADOR DE ELEMENTOS DENTRO DEL ARRAY. si no le pones nada, te va a hacer un array con un elemento solo

let stringHola = "hola como estas" stringHola.split() ========= stringHola== [hola como estas]

let stringHola = "hola como estas" stringHola.split(" ") ========= stringHola=== [hola] [como] [estas]
