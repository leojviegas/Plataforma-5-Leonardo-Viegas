TIPOS DE FUNCIONES

let, const, var

let= se puede modificar (EJ: yo podria hacer let nombre = 2; nombre = nombre + 1; ahora la variable llamada "nombre" vale "3"

const = no se puede modificar (EJ: yo NO podría hacer nombre = nombre + 1, porque la estaría modificando

var = se puede modificar, y aparte se puede declarar dos veces la misma variabe. Por eso, NO la vamos a utilizar (cada vez está quedando más en desuso)

PROMPT

el prompt es para pedirle al usuario que meta una data, que luego se puede usar como variable
